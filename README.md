# 3D Default Compat

Adds 3D textures to modded blocks

## ✨ Compatibility

- [Soul Candles](https://modrinth.com/project/XHXMTS5X)

## 🚀 Planned Features

- Biomes O’ Plenty
- Critters & Companions
- Deeper & Darker
- Enhanced Workbenches
- Friends & Foes
- Pyrellium
- Wilder Wild

## 📦 Installation

You can download 3D Default Compat on [Modrinth](https://modrinth.com/project/yHnAkckW) or [CurseForge](https://www.curseforge.com/minecraft/texture-packs/3d-default-compat)

## ⚙️ Requirements

You need the following resource pack for it to work properly:

- [3D Default](https://modrinth.com/project/5aPp18Lx)
